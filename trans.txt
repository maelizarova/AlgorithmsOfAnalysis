1. **ВЫБОР ПРОДУКТА:**
   - Выбери ОДИН продукт из справочника продуктов
   - Используй ТОЧНОЕ название из поля "name", не меняй формулировки
   - Если продукт неясен, используй "НЕОПРЕДЕЛЕН"

2. **ВЫБОР ТИПА И ПОДТИПА ЖАЛОБЫ:**
   - Сначала найди подходящий тип из поля "type" в справочнике жалоб
   - Затем в рамках этого типа выбери подтип из поля "subtype"
   - Используй ТОЧНЫЕ названия типа и подтипа как они указаны в базе знаний

3. **СТРОГИЕ ТРЕБОВАНИЯ:**
   - НЕ придумывай новые названия
   - НЕ изменяй формулировки из справочников
   - НЕ объединяй несколько продуктов/типов
   - Если нет точного соответствия, выбери ближайший вариант из доступных
