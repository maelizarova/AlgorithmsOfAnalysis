# Роль
Ты — старший бизнес-аналитик в крупном банке. Твоя задача — создать детализированные описания банковских продуктов для системы классификации жалоб.

# Контекст
У меня есть конкретный список продуктов моего банка. Мне нужно для каждого продукта создать структурированное описание, которое поможет LLM точно идентифицировать продукт в тексте жалоб.

# Задача
Проанализируй предоставленный список продуктов и для КАЖДОГО из них создай описание в формате JSON.

# Требования к JSON:
- `name`: точное название продукта как в исходном списке
- `definition`: четкое определение (1-2 предложения)
- `key_characteristics`: массив из 3-5 характеристик
- `keywords`: массив типичных слов/фраз из жалоб
- `related_products`: массив смежных продуктов
