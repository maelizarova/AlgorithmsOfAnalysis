В тексте перечислины возможные проблемы, с которыми сталкиваются клиенты при обслуживании в банке, при работе с дебетовыми картами. Проблем много, многие из них повторяются, или написаны разными словами про одно и тоже. Нужно сгруппировать все проблемы по смыслам, чтобы качественно увидеть ситуацию с клиентским опытом со всех сторон. Классификация должна быть двух уровневой, с небольшим пояснением что куда относится, при этом все проблемы должны быть логически разделенными, без двойных интерпретаций, четкими

Инструкция:

Сгруппируй все перечисленные проблемы клиентов банка, связанные с обслуживанием по дебетовым картам, по смыслу.
Сформируй двухуровневую классификацию: category (главная тема) и sub‑category (конкретный аспект).
Для каждой подкатегории укажи краткое описание проблемы (1‑2 предложения).
Убедись, что каждую проблему можно отнести только к одной подкатегории (исключи двойные интерпретации).
Выведи результат в виде таблицы с колонками: Category | Sub‑category | Description.
Пример формата:

| Category | Sub‑category | Description |

|----------|--------------|------------|
